# E-commerce CMS Web App Server
E-commerce CMS (content management system)

# URL
  http://localhost:3000

# Method

## Available Endpoints List
- `POST /login`
- `POST /register`

- `GET /products`
- `POST /products`
- `PUT /products/:id`
- `DELETE /products/:id`

- `POST /cart`
- `GET /cart`
- `POST /cart/checkout`
- `DELETE /cart/:id`
- `GET /cart/history`

> User login

### *Request Header*
```Not Needed```
### *Request Body*
 ```json
{
  "email": "admin@mail.com",
  "password": "qweqwe"
}  
```
